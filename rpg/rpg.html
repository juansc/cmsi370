Creating a New Character

Click on the add button on the character list. You can create a custom character of your choosing or you can choose to spawn a random character into your list.

Editing a Character

Make sure the character is selected. When a character is selected its row will be blue and the character card will display the character's information. If you click the Edit button a dialog box will appear that will let you change the character's attributes.

Once you have finished making changes, press Save. Press Cancel to not save the changes you made.

Deleting A Character

Select the character you wish to delete. When a character is selected its row will be blue and the character card will display the character's information. Click on the Delete button. A dialog box will ask you if you're sure you want to delete your character. Click Yes to delete and No if you wanna save your poor character.
